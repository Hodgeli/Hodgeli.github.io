Array Partition I

给定一个2n整数的数组，你的任务是把这些整数分成n对整数，比如（a1，b1），（a2，b2），…，（an，bn），这使得min（ai， bi）的和尽可能大。

思路

排序，再以步长为2遍历数组，取每组中的较小值相加。

Toeplitz Matrix

现在给出一个M×N矩阵，当且仅当矩阵是Toeplitz时才返回True。
